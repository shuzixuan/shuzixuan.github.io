2. Mesh Analysis 网孔分析法（网孔电流法）

Definition :
    A loop is independent : It contains at least one branch which is not a part of any other independent loop.
    A mesh is a loop that does not contain any other loop within it.
eg :

  1. All possible independent loop combination : 1,2 1,3 2,3
  2. Mesh : 1,2
  3. mesh $\in$ independent loop

  • Loop analysis 回路电流法
      Step1 Assign $i_1,i_2…i_m$ to the m independent loop
      Step2 Apply KVL to the m loops
    $\downarrow$ for planar circuit

  • Mesh analysis 网孔电流法
      Step1 Assign $i_1,i_2…i_m$ to the m meshes
      Step2 Apply KVL to the m loops
